A responsive, user-friendly web application to manage personal finances efficiently. Built with HTML, CSS, and JavaScript, it allows users to track income and expenses, calculate balances in real-time, and persist data using Local Storage.

Key Features
Add Transactions: Record income or expenses with descriptive names and amounts.
Dynamic Balance: Automatically updates total balance as transactions are added or removed.
Transaction History: View a detailed log of all transactions.
Delete Functionality: Easily remove unwanted or incorrect entries.
Persistent Storage: Transactions are saved in Local Storage — data remains after page reloads.
Input Validation: Ensures only valid and complete entries are added.
